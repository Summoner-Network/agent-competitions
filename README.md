## Season 1

### Rules

- **Entry file name:** your agent must be in a file named `agent.py`.
- **Entry point:** your agent must expose an async function:
  ```py
  async def run_agent(payload: dict):
    ...
  ```
- **Response time:** your agent must produce a response within **45 seconds** of receiving an input `payload`.
